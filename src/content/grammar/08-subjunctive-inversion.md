## 假設語氣（Subjunctive Mood）

假設語氣出現在特定的固定句型中，動詞形式不跟著主詞變化，固定用**原形動詞**。

### 第一類：It is + 形容詞 + that + 主詞 + **原形動詞**

這類句型表達「必要性、建議、重要性」，that 子句裡的動詞用**原形**，不加 s，不用 was。

常見的形容詞：important / essential / necessary / crucial / recommended / required / suggested / proposed

- It is essential **that** every participant **sign** the consent form.（不是 signs）
- It is crucial **that** the data **be** collected carefully.（不是 is）
- It is recommended **that** the sample **remain** at room temperature.（不是 remains）

加否定：直接在原形前加 not：

- It is important **that** participants **not be** informed of the hypothesis.

### 第二類：動詞 + that + 主詞 + **原形動詞**

這類動詞表達「要求、建議、堅持、命令」：
suggest / recommend / require / demand / insist / propose / request / urge / order

- The committee **recommended** that the project **be** postponed.
- The doctor **insisted** that the patient **rest** for a week.
- The protocol **requires** that all samples **be** frozen immediately.

> 這兩類假設語氣在 TOEFL 閱讀不常考，但寫作用對了能大幅提升文法分數。記住：**要求類動詞 + that + 原形**。

## 倒裝句（Inversion）

英文正常語序是「主詞 + 動詞」，倒裝就是把動詞或助動詞移到主詞前面。

### 為什麼用倒裝？

1. **強調某個副詞或否定詞**（放句首 → 後面倒裝）
2. 增加文章的正式感與語氣
3. TOEFL 閱讀中偶爾出現，需要能正確解讀句意

---

### 否定副詞開頭 → 倒裝

把否定或限制副詞移到句首，後面**助動詞 + 主詞 + 動詞**：

- **Never** have scientists observed such a rapid change in climate.
  （正常：Scientists have never observed...）

- **Rarely** does the phenomenon occur under natural conditions.
  （正常：The phenomenon rarely occurs...）

- **Seldom** did researchers encounter such anomalies.
  （正常：Researchers seldom encountered...）

- **Not only** did the study reveal new data, **but** it also challenged existing theories.
  （正常：The study not only revealed...）

- **Only then** did the team understand the significance of the findings.
  （正常：The team understood only then...）

---

### only + 副詞片語開頭 → 倒裝

- **Only after** the results were confirmed **did** the team publish the paper.
- **Only when** temperatures dropped below zero **did** ice form on the surface.
- **Only by** combining multiple methods **can** researchers achieve reliable results.

---

### So / Neither / Nor 開頭的倒裝（表達「也一樣」）

- A: The sample was contaminated.
- B: **So was** the control group.（control group 也一樣被污染了）

- A: The first experiment failed.
- B: **So did** the second one.

- A: The results were not conclusive.
- B: **Neither were** the preliminary findings.（也一樣不確定）

---

### No sooner...than / Hardly...when

這兩個結構表示「一…就…」，前半句倒裝：

- **No sooner had** the experiment begun **than** an error was detected.
  （正常：The experiment had no sooner begun than...）

- **Hardly had** the team published the results **when** criticisms emerged.
  （正常：The team had hardly published...）

---

## 快速辨認倒裝句

看到句子開頭是這些詞，後面十之八九是倒裝：

- Never / Rarely / Seldom / Hardly / Scarcely
- Not only / Not until / Not since
- Only after / Only when / Only by / Only then
- So + adj（So great was the impact that...）
- No sooner... than / Hardly... when

**辨認方法**：找主詞在哪裡。助動詞（do/did/have/has/is/are/was/were）在主詞前面 → 就是倒裝句。

- *Rarely **does** the system **fail** under normal conditions.*
  （助動詞 does 在主詞 the system 前 → 倒裝）
