## For Scott & Chloe
請至 `/src/content/reading/common` 新增你們的 `md` file。  
檔名請用輪到你報告英文該天的日期，ex: `20260222.md`，表示為 2026 年 02 月 22 號。

文章標頭請遵守以下格式：
```
---
date: 2026/01/21
title: Seven travel trends that will define 2026
link: https://www.bbc.com/travel/article/20251201-seven-travel-trends-that-will-define-2026
author: 'chloe'
---
```
